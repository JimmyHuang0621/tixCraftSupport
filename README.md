# 拓元輔助程式

## 說明
1. 自動點選"立即購票"，顯示節目列表。
2. 在節目列表中，自動尋找節目日期並點選"立即訂購"。

<img width="768" src="assets/step1.png" alt="step1" />

3. 手動選擇座位 (尚未實作)
4. 自動選擇最大張數與勾選同意條款，並準備好輸入驗證碼，輸入完按Enter。

<img width="768" src="assets/step2.png" alt="step2" />


## 設定
1. 右鍵點選拓元輔助ICON，選擇選項。
2. 點選日期

<img width="768" src="assets/options.png" alt="options" />
